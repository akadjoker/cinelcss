Introdução ao CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilo usada para estilizar elementos HTML. Aqui estão alguns conceitos básicos:

Seletores

Os seletores CSS são usados para selecionar os elementos HTML aos quais desejas aplicar estilos. Por exemplo:

    
        /* Seleciona todos os parágrafos */
        p {
            color: blue;
        }

        /* Seleciona todos os elementos com a classe "destaque" */
        .destaque {
            font-weight: bold;
        }

        /* Seleciona o elemento com o id "cabecalho" */
        #cabecalho {
            font-size: 24px;
        }

        /* Seleciona links que estejam dentro de um parágrafo */
        p a {
            text-decoration: underline;
        }

Propriedades e Valores

No CSS, as propriedades são os atributos que desejas alterar, e os valores são os diferentes estilos que podes aplicar a essas propriedades. Por exemplo:

    
        /* Altera a cor do texto para vermelho */
        color: red;

        /* Define a largura da borda como 1px */
        border-width: 1px;

        /* Define o fundo como amarelo */
        background-color: yellow;

        /* Define a fonte como Arial */
        font-family: Arial;

        /* Define o tamanho da fonte */
        font-size: 16px;

        /* Define a altura da linha */
        line-height: 1.5;

        /* Define a margem */
        margin: 10px;

        /* Define o preenchimento */
        padding: 10px;

        /* Define a largura */
        width: 200px;

        /* Define a altura */
        height: 100px;

        /* Define a opacidade */
        opacity: 0.5;

        /* Define o alinhamento de texto */
        text-align: center;

        /* Define a decoração do texto */
        text-decoration: underline;

        /* Define a sombra do texto */
        text-shadow: 2px 2px 4px #000;

        /* Define a rotação */
        transform: rotate(45deg);

        /* Define a transição */
        transition: all 0.3s ease;

        /* Define a visibilidade */
        visibility: hidden;

        /* Define a z-index */
        z-index: 999;

        /* Define o estilo de exibição */
        display: block;

        /* Define a quebra de linha */
        word-wrap: break-word;

        /* Define a direção do texto */
        direction: rtl;

        /* Define a posição */
        position: relative;

        /* Define a flutuação */
        float: left;

        /* Define a cor da borda */
        border-color: black;

        /* Define o estilo da borda */
        border-style: dashed;

        /* Define o raio do canto da borda */
        border-radius: 5px;

        /* Define o espaçamento entre letras */
        letter-spacing: 2px;

        /* Define a transformação de texto em maiúsculas */
        text-transform: uppercase;

        /* Define o espaçamento entre linhas */
        line-height: 1.5;

Modelo de Caixa (Box Model)

O modelo de caixa é uma representação visual de como os elementos HTML são dimensionados e posicionados. Ele consiste em quatro partes: margem, borda, preenchimento e conteúdo.

Posicionamento

A propriedade position do CSS determina como um elemento é posicionado em uma página web. Os valores possíveis são static, relative, absolute e fixed.

Flexbox

O flexbox é um modelo de layout unidimensional que permite criar layouts mais complexos e flexíveis. Ele é especialmente útil para alinhar elementos em uma linha ou em uma coluna.

Grid

O grid é um sistema bidimensional de layout que permite criar layouts complexos e responsivos. Ele divide o espaço em linhas e colunas, permitindo um controle preciso sobre o posicionamento dos elementos.

Seletores Avançados

Além dos seletores básicos, o CSS oferece seletores mais avançados que permitem selecionar elementos com base em atributos, estados ou posições específicas.

Media Queries

As media queries são usadas para aplicar estilos específicos com base nas características do aparelho ou do navegador, como largura do monitor, orientação, resolução, etc. Isso é essencial para criar designs responsivos que se adaptem a diferentes dispositivos e tamanhos de monitores.
