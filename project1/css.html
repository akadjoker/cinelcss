CSS

CSS (Cascading Style Sheets) é uma linguagem de folha de estilo usada para estilizar elementos HTML. Com CSS, podes controlar a aparência de textos, cores, margens, posicionamento e muito mais.

Existem várias maneiras de adicionar estilos CSS a uma página da web:

  • Estilos embutidos: usando a tag <style> no cabeçalho da página
  • Estilos internos: usando a tag <style> dentro de um elemento <style> na própria página
  • Estilos externos: usando um arquivo CSS externo vinculado à página usando a tag <link>

Seletores Básicos

Os seletores CSS são usados para selecionar e estilizar elementos HTML. Alguns seletores básicos incluem:

  • Elementos: `body`, `p`, `h1`, `div`, etc.
  • Classes: `.classe`, `.destaque`, etc.
  • IDs: `#id`, `#header`, etc.

Exemplo:

          

              p {
                  color: #333;
              }


              .destaque {
                  font-weight: bold;
                  color: blue;
              }


              #header {
                  background-color: #333;
                  color: #fff;
                  padding: 20px;
              }

Box Model

O Box Model é uma parte fundamental do layout CSS e consiste em quatro propriedades: margin, border, padding e content.

Exemplo:

          

              .box {
                  margin: 10px;
                  border: 1px solid #333;
                  padding: 20px;
                  background-color: #f4f4f4;
              }
