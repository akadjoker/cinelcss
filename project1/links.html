Links

Links Internos

Links internos são usados para navegar dentro do mesmo site. Eles são criados com a tag <a> e o atributo href contendo o caminho para a página desejada.

Sobre HTML

Links Externos

Links externos são usados para navegar para outros sites. Eles são criados da mesma forma que os links internos, mas com URLs externas.

Links Âncora

Links âncora são usados para navegar para seções específicas na mesma página. Eles são criados da mesma forma que os links internos, mas com o atributo href contendo o ID da seção desejada.
