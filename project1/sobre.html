Sobre HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar e projetar páginas da web. Ele define a estrutura e o conteúdo de uma página da web usando uma série de elementos e tags.

Para começar a criar páginas da web em HTML, você precisa entender alguns conceitos básicos, como:

  • Tags HTML básicas: <html>, <head>, <title>, <body>
  • Elementos de texto: <h1>, <p>, <strong>, <em>
  • Links: <a>
  • Imagens: <img>
  • Listas: <ul>, <ol>, <li>
